PROJECTS

Color Switch Game

A remake of the original Color Switch game with additional features like saving the game at any point and make local accounts. The game is made entirely in JavaFX. While the actual game has a lot of modes, we have only implemented the endless mode. Difficulty increases gradually and collected stars can be used to revive. Collaborated with Sarthak Johari.

Handy Write: Audio / Visual learning tool

Our first major project. It is an interactive tool aimed towards kids to teach them to recognise and write letters and words using audio and visual aids. We also used Arduino to manage sensors that enhanced the overall experience of the user. This was our first project and so the code is a little messy. Collaborated with Samyak Jain and Dhroov Goel.

Command Line Shell

A miniature command line shell that is able to perform certain basic internal and external commands like cd, ls, mkdir, etc using system calls. Written in C. Each command also implements two options. Even though this is by no means a replacement for an actual shell, it is still a fun little project.
